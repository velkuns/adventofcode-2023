# Advent Of Code - 2023


## Install

```bash
$ composer install
```

## Solve puzzle day
```bash
Use    : bin/aoc solve [OPTION]...
OPTIONS:
          --help                        Reserved - Display Help
  -d ARG, --day=ARG                     Day to solve (default current day)
  -e,     --example                     Example Only
  -s,     --skip-empty-lines            Remove empty lines from inputs

```

Example:
```bash
$ bin/aoc solve --day=1   
------------------------------------------ EXAMPLES ------------------------------------------
*  :  11 - expected: 11
** :  6 - expected: 6
------------------------------------------- OUTPUT -------------------------------------------
*  : 6726 - [0s] - [2.2MB]
** : 3316 - [0s] - [2.2MB]
```
